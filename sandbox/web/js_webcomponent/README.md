# Bold Neo-Brutalist Web Component Example

This project demonstrates a striking, high-contrast web component built with Vanilla JavaScript and CSS, following a Neo-Brutalist design aesthetic.

## Design Principles
- **Bold Typography**: Heavy use of 'Outfit' variable font with weights up to 900.
- **Neo-Brutalist Aesthetic**: Plain colors, thick black borders, and hard offset shadows.
- **High Contrast**: Stark black and white base with vibrant primary color accents.
- **Shadow DOM**: Fully encapsulated component logic and styles.

## How to Run

1. Ensure you have `go-task` installed (optional, but recommended).
2. Run the project:
   ```bash
   task run
   ```
   Or use npx directly:
   ```bash
   npx serve .
   ```

## Project Structure
- `index.html`: Main showcase page with a bold, structured layout.
- `style.css`: Global design tokens (borders, shadows, typography).
- `components/BoldCard.js`: The `bold-card` web component implementation.
- `Taskfile.yml`: Task definitions.
