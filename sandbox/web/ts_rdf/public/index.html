Introduction to Triples

Step Active

RDF stores data as triples: Subject → Predicate → Object. Think of it as a set of facts forming a web of data.

Source Snippet

import * as N3 from 'n3';
const store = new N3.Store();
// Triple: Alice knows Bob
store.addQuad(
  namedNode(':Alice'), 
  namedNode('foaf:knows'), 
  namedNode(':Bob')
);
