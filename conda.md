init conda in powershell
```powershell
conda init powershell
```

create env in conda
```powershell
conda create --name {name} python=X.X
```

commands conda
```powershell
conda activate {name}
```

```powershell
conda deactivate
```

```powershell
conda env list
```

```powershell
conda env remove --name {name}
```
